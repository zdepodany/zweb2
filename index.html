V čem pracuji

Díky kombinaci těchto ověřených nástrojů se Vaše vize stane realitou.

Figma

Vizuál krok za krokem. Za pomocí Figmy vytvořím Vašemu produktu dokonalý design na míru.

Cursor

Moderní vývojové prostředí s AI asistencí pro efektivní a rychlý vývoj. Zajišťuje vysokou produktivitu a kvalitu kódu.

Webflow

Základ pro moderní web. Za pomocí Webflow vám vytvořím konzistentní a funkční webové stránky.

HTML

Nosné a stabilní základy. Díky nim Váš web bude rychlý, spolehlivý a optimalizovaný pro vyhledávače.

CSS

Vzhled, co během prvních tří vteřin zaujme. Díky CSS adaptuji design na Vaše webové stránky.

JavaScript

Programovací jazyk JavaScript je nepostradatelným nástrojem každého profesionála. Pro všechny stránky zajišťuje dynamiku a interaktivitu.
